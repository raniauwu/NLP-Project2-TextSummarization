# 📰 Project 2: Text Summarization with BERT

**NLP Bootcamp - Indonesia AI Career Bootcamp**  
Dataset: Liputan6 (224,637 artikel berita)  
Model: IndoBERT + bert-base-multilingual-cased  
Evaluation: ROUGE-1, ROUGE-2, ROUGE-L  

## Highlights
- Extractive summarization approach
- Stratified downsampling (60% train)
- IndoBERT ROUGE-1: **0.2711** (outperformed multilingual BERT)
